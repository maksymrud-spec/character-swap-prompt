# Changelog

## [1.0.0] — initial version

- First versioned release of the character-swap-prompt skill
- Supports two-pack input: reference characters (white bg) + scene images
- Generates single English prompt in Nano Banana 2 format
- Handles 1–3+ character replacements
- @name extracted from filename automatically
- Edge case handling: ambiguous positions, unmatched characters, missing names
